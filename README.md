# 📚 개인 과제 관리 프로그램 (Personal Assignment Management Program)

본 프로젝트는 대학생 및 학습자의 학업 일정을 효율적으로 관리할 수 있도록 지원하는 **개인 과제 관리 프로그램**입니다.
이 리포지토리는 PRD(제품 요구사항 정의서)에 명시된 파이썬(Python) 기반 터미널 관리 프로그램과, GitHub Pages에 배포하여 편리하게 사용할 수 있는 프리미엄 웹 대시보드(HTML/CSS/JS) 버전을 동시에 포함하고 있습니다.

---

## 🛠️ 기능 요약
- **과제 등록**: 과목명, 과제명, 마감일(YYYY-MM-DD 형식)을 입력받아 과제를 등록합니다.
- **과제 조회**: 등록된 과제의 목록과 마감일까지 남은 기한(D-Day), 현재 진행 상태를 직관적으로 확인할 수 있습니다.
- **과제 검색**: 특정 과목명이나 과제명을 기반으로 과제를 필터링하여 검색합니다.
- **진행 상태 관리**: 과제별로 `'시작 전'`, `'진행 중'`, `'완료'` 상태를 업데이트할 수 있습니다.
- **과제 수정 & 삭제**: 과제 정보의 내용을 수정하거나 목록에서 삭제할 수 있습니다.
- **통계 요약**: 전체 과제 수, 시작 전 과제 수, 진행 중 과제 수, 완료된 과제 수의 실시간 통계를 시각적으로 제공합니다.
- **데이터 보존**: 
  - **파이썬**: 로컬 `assignments.json` 파일에 저장되어 다시 실행해도 데이터가 보존됩니다.
  - **웹**: 브라우저 로컬 저장소(`localStorage`)에 자동 저장되어 페이지를 닫거나 새로고침해도 데이터가 보존됩니다.

## 💻 1. 파이썬 CLI 프로그램 실행 방법 (Terminal / Colab)

### 실행 단계
1. 컴퓨터 터미널(Command Prompt, PowerShell 등)을 실행합니다.
2. 아래 명령어로 파이썬 스크립트를 실행합니다:
   ```bash
   python assignment_manager.py
   ```
3. 안내되는 메뉴 번호(1~8)를 입력하여 과제를 관리합니다.

### Google Colab에서 실행 방법
1. Google Colab 노트북을 새로 생성합니다.
2. `assignment_manager.py` 파일의 전체 코드를 복사하여 Colab의 코드 셀에 붙여넣습니다.
3. 셀을 실행하여 화면에 나타나는 메뉴 기반의 인터페이스를 통해 과제를 등록하고 관리합니다.

## 🌐 2. 웹 대시보드 사용 및 배포 방법 (GitHub Pages)

### 로컬에서 실행 방법
1. 이 폴더에 있는 `index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge 등)에서 즉시 실행합니다.
2. 추가 설치가 전혀 필요 없이 사용 가능합니다.

### GitHub Pages 배포 가이드
수업 기말 과제 대체 제출을 위해 GitHub Pages에 호스팅하는 방법은 다음과 같습니다:
1. GitHub 리포지토리(Repository)를 새로 생성합니다.
2. 이 프로젝트의 모든 파일(`index.html`, `styles.css`, `app.js`, `README.md` 등)을 리포지토리에 업로드(Push)합니다.
3. 리포지토리의 **Settings (설정)** 탭으로 이동합니다.
4. 왼쪽 사이드바에서 **Pages**를 클릭합니다.
5. **Build and deployment** 하단의 **Branch**를 `main` 또는 `master`로 지정하고, 폴더는 `/ (root)`로 선택한 뒤 **Save** 버튼을 누릅니다.
6. 약 1~2분 후 생성되는 `https://<사용자이름>.github.io/<리포지토리이름>/` 링크를 통해 접속하여 사용하고, 이 링크를 기말 과제물로 제출합니다.
